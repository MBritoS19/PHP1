1. Formatando números

number_format permite tratar valores reais em PHP. 

number_format(número, decimais, separador, separa_milhar)
número
número que será formatado.
decimais
número de casas decimais.
separador
caractere que será usado para separar decimais. Padrão ".".
separa_milhar
caractere que será usado para separar milhar. Padrão ",".
echo number_format(3.14159, 2, ",",); // 3,14

2. Convertendo Strings em Números em PHP

2.1. Conversão explícita

Type casting converte uma String para o tipo indicado entre parênteses. 

$string_numero = "123";
$numero_inteiro = (int) $string_numero; // 123 (inteiro)
$numero_float = (float) $string_numero; // 123.0 (float)

2.2. Funções para conversão

As funções intval e floatval. O PHP tenta encontrar um valor no início da String: 

$var = '122.34The';
$float_var = floatval($var);
echo $float_var;

Se The estiver no início, o resultado é 0: 

$var = 'The122.34';
$float_var = floatval($var);
echo $float_var;

O que acontece aqui? 

$var = '122,34The';
$float_var = floatval($var);
echo $float_var;

2.3. Conversão em uso

String usada em operação matemática: 

echo "2" * 3;

Atenção para conversões impossíveis: 

echo "a2" * 3;

Gerando algo como: Fatal error: Uncaught TypeError: Unsupported operand types: string + int

3. Método de envio

Podemos usar $_SERVER para verificar o método de envio de um formulário com $_SERVER['REQUEST_METHOD']. E também para imprimir o IP do usuário: $_SERVER['REMOTE_ADDR'].

3.1. Exemplo de formulário e recebimento:

<!DOCTYPE html>
<html>
<head>
    <title>Formulário Simples</title>
</head>
<body>
    <form action="recebe.php">
	<label for="nome">Nome:</label>
	<input type="text" id="nome" name="nome">
	<br>
	<input type="submit" value="Enviar">
    </form>
</body>
</html>

3.2. Exemplo de teste

<?php
// Verifica se o formulário foi enviado via GET
if ($_SERVER['REQUEST_METHOD'] === 'GET') {
    // Obtém o nome enviado pelo formulário
    $nome = $_POST['nome'];

    // Imprime o nome
    echo "Olá, " . $nome . "!";
} else {
    // Se não foi enviado via GET, exibe mensagem
    echo "Favor preencher o formulário.";
}
?>

3.3. Burlando

O usuário poderia tentar enviar dados diretamente para a URL de recebe.php com parâmetros na URL.

4. Objeto

Dada a classe produto. 

class Produto {
    public $nome;
    public $descricao;
    public $valor;
    public $imagem;

    // Construtor para inicializar os atributos
    public function __construct($nome, $descricao, $valor, $imagem) {
	$this->nome = $nome;
	$this->descricao = $descricao;
	$valor = (float) $valor; // "5.3" -> 5.3
	$this->valor = $valor;
	$this->imagem = $imagem;
    }

    // Método para exibir as informações do produto
    public function exibirInformacoes() {
	echo "Nome: " . $this->nome . "<br>";
	echo "Descrição: " . $this->descricao . "<br>";
	echo "Valor: R$" . number_format($this->valor, 2, ',', '.') . "<br>";
	echo "Imagem: " . $this->imagem . "<br>";
    }
}
