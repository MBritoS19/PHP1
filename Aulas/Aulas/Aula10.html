1. Sistema de notas

1.1. Criação de bancos de dados e tabelas.

Utilizando phpmyadmin para criar banco e tabelas.

1.1.1. Exercícios

  1. Crie o banco de dados chamado sistema_notas.
  2. Crie a tabela turmas (id, nome)
  3. Crie a tabela alunos (id, nome, id_turma).
  4. Crie a tabela notas (id, valor, id_aluno, id_turma).
  5. Exporte o SQL do banco completo.

1.2. Cadastro de turmas

Criação de formulários HTML e inserção de dados em um banco de dados utilizando PHP.

1.2.1. Exercícios

  1. Criar um formulário HTML para cadastrar novas turmas.
  2. Criar página PHP para inserir os dados da turma no banco de dados. Implementar validação dos dados para garantir que o nome da turma seja preenchido.

1.3. Cadastro de alunos

Criação de formulários HTML e inserção de dados em um banco de dados utilizando PHP.

Vamos usar o elemento <select> para mostrar as turmas cadastradas.

1.3.1. Exercícios

  1. Criar um formulário HTML para cadastrar novos alunos. Usar <select> para a seleção da turma.
  2. Utilizar PHP para inserir os dados do aluno no banco de dados. Implementar validação dos dados para garantir que o nome do aluno seja preenchido e que a turma exista.

1.4. Cadastro de notas

Trabalhar com a tabela de notas

1.4.1. Exercícios

  1. Criar um formulário HTML para cadastrar as notas dos alunos.
  2. Utilizar PHP para inserir os dados das notas no banco de dados. Implementar validação dos dados para garantir que a nota esteja dentro do intervalo válido (por exemplo, de 0 a 10). Verificar se o aluno e a turma existem antes de inserir a nota.

1.5. Consultas e Listagens

Aplicar consultas SQL para recuperar dados e exibir os resultados de forma organizada.

Exibir os resultados em uma tabela HTML.

1.5.1. Exercícios

  1. Criar uma página para listar todos os alunos de uma turma específica.
  2. Criar uma página para mostrar as notas de um aluno selecionado.
