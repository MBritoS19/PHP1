1. Buscando dados

Vamos usar o phpmyadmin para gerar o código SQL que retorna os dados de uma tabela: 

SELECT `nome`, `quantidade` FROM `ingredientes`

Antes de exibir os dados precisamos preparar e executar o comando SQL: 

// comando que será executado
$comandoSQL = 'SELECT `nome`, `quantidade` FROM `ingredientes`';

// prepara para executar o comando
$comando = $conexao->prepare($comandoSQL);

// executa o comando
$comando->execute();

Podemos usar o método fetch() buscar os dados de uma consulta SQL:

A cada vez que usamos fetch() uma linha do resultado é buscada: 

$linha = $comando->fetch();

echo $linha['nome'];
echo "<br>";
echo $linha['quantidade'];
echo "<br>";

Quando as linhas do resultado terminam, fetch() retorna false, por isso podemos usá-lo em um laço de repetição: 

while($linha = $comando->fetch()) {
  echo $linha['nome'];
  echo "<br>";
  echo $linha['quantidade'];
  echo "<br>";
}

2. Apagar

Para apagar algo de uma tabela SQL usamos o comando DELETE. Vamos ver como funciona:

2.1. Com formulário

Vamos receber o valor de alguma coluna que será retirada: 

"DELETE FROM ingredientes WHERE `ingredientes`.`id` = 7"

Usaremos o formulário: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Recebe id</title>
</head>
<body>
    <form action="apaga.php">
	<label for="">ID que será apagado:</label>
	<input type="text" name='id'><br>
	<input type="submit">
    </form>
</body>
</html>

A página que apaga o item com o número recebido deve conter: 

<?php
    $servidor = 'localhost';
    $banco = 'receitas';
    $usuario = 'root';
    $senha = '';

    $conexao = new PDO("mysql:host=$servidor;dbname=$banco", $usuario, $senha);

    $comandoSQL = "DELETE FROM ingredientes WHERE `ingredientes`.`id` = :id";

    try {
	$comando = $conexao->prepare($comandoSQL);

	$resultado = $comando->execute(array('id' => $_GET['id']));

	if($resultado) {
	    echo "Item apagado!";
	} else {
	    echo "Erro ao apagar o item!";
	}
    } catch (Exception $e) {
	echo "Erro $e";
    }
?>

2.3. Com link

Ao enviar os dados, é como se o usuário digitasse um endereço como: 

http://localhost/receitas/apaga.php?id=6

Se o usuário acessar: 

http://localhost/receitas/apaga.php?id=8

O sistema vai tentar apagar o item com id=8. Podemos então criar um link com o endereço e quando o usuário acessar o link ele apaga o item. 

<a href="http://localhost/receitas/apaga.php?id=8">Apagar</a>

Vamos então gerar o link automaticamente em mostra.php: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mostrando dados</title>
</head>
<body>
    <?php
$servidor = 'localhost';
$banco = 'receitas';
$usuario = 'root';
$senha = '';

$conexao = new PDO("mysql:host=$servidor;dbname=$banco", $usuario, $senha);

$comandoSQL = 'SELECT * FROM `ingredientes`';

$comando = $conexao->prepare($comandoSQL);
$resultado = $comando->execute();

if($resultado) {
    echo "Mostrando resultado:<br>";
    while($linha = $comando->fetch()) {
	echo $linha['nome'];
	$id = $linha['id'];
	echo "<a href='apaga.php?id=$id'>Apagar</a><br>";
    }
} else {
    echo "Erro no comando SQL";
}
    ?>

</body>
</html>

E usar o método exec() ao invés de execute() em apaga.php: 

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Apaga item</title>
</head>
<body>
<?php
    $id = $_GET['id'];
    $comandoSQL = "DELETE FROM ingredientes WHERE `ingredientes`.`id` = $id";

    $servidor = 'localhost';
    $banco = 'receitas';
    $usuario = 'root';
    $senha = '';

    $conexao = new PDO("mysql:host=$servidor;dbname=$banco", $usuario, $senha);

    try {
	$resultado = $conexao->exec($comandoSQL);

	if($resultado != 0) {
	    echo "Item apagado!";
	} else {
	    echo "Erro ao apagar o item!";
	}
    } catch (Exception $e) {
	echo "Erro $e";
    }
?>
</body>
</html>
